退費政策

一、付款政策
(一) 採用信用卡方式付款時，請你填寫確實的信用卡資料，若有不實或任何未經持卡人許可有盜刷信用卡情形時，本公司得以暫停或永久終止你的購買資格，並停止你尚未使用的兌換碼憑證，如確有違法情事，本公司得配合司法警察機關提供相關訊息及依法追究，打擊盜刷決不寬貸。
(二) 你瞭解且同意，所有經由所使用本服務進行付款時，無須使用簽帳單、當場簽名或另簽署書面契約，你不得僅以未簽名為理由，拒繳應付款項。

二、退款政策
(一) 退費規則
• 交易成功後，若符合全額退款條件期間申請取消，則此訂單將會全額退費至原信用卡帳戶，並退還原使用之 Fun 幣及優惠碼。惟優惠碼已逾期失效者，恕無法補發。
• 交易成功後，若符合部份退款條件期間申請取消，則此訂單將會按該商品取消與更改時間政策將應退款金額退費至原信用卡帳戶，並將退還原使用之 Fun 幣，若該訂單使用之優惠碼恕無法返還。
• 預訂完成後因店家已為您保留服務，若超過該商品可更改訂單時限，恕無法修改或退款。超過時效之訂單亦無法修改或退款。
• 特殊商品、檔期活動有其他特殊規範者，則依該商品內容、取消與更改時間政策所標示之規則為準。

(二) 特殊情形補充
• 若服務提供業者有其他特殊規範(例如：特殊假期無法退訂、特殊活動十日內無法取消等)將標示於商品內容、取消與更改時間政策上，並且在您訂購之前提醒，取消規則將依據每項商品內容、取消與更改時間政策之說明及提醒可能有不同。
• 若遇到不可抗拒之因素(例如地震、颱風、戰爭) 等特殊情況要取消訂單，您將有機會收到全額退款。
• 您提出預訂請求時，即代表您同意本平台及服務提供業者的退款政策。
• 由於本平台所銷售之商品及服務皆為服務提供業者即時提供並且保證訂位，因此在您購買商品或服務後，若未於預訂時段出席且事前未與客服聯繫告知，則服務提供業者有權收取全額費用、不提供退費請求。
• 信用卡刷退將退回原付款之信用卡中，您的銀行或信用卡公司約需14個工作天處理；因信用卡結帳日期依各發卡銀行而異，刷退款項可能會顯示在次月的帳單中，欲查看退款狀態，請自行與您的銀行或信用卡公司聯絡。

(三) 禮物商品退費規則
• 收禮人選擇禮物商品時間完成預訂後，禮物即已領取並歸為收禮人所有，送禮人即不可取消或撤回此禮物。
• 收禮人雖在選擇禮物商品時間完成預訂後為禮物的所有人，但一旦於 FunNow 完成預訂後即不可取消或要求退款。
• 若 (1) 收禮人未於送禮人購買禮物商品後 3 天內打開禮物盒 ; (2) 收禮人於打開禮物盒後 14 天內未選擇禮物商品服務時間完成預訂 ; (3) 送禮人於收禮人選擇禮物商品服務時間完成預訂前主動取消該禮物商品；(4) 送禮人購買的禮物商品因店家下架或該店家歇業，送禮人購買禮物商品之金額將根據使用之支付工具全額退款，並退還使用之 Fun 幣和優惠碼。請注意，優惠碼使用期限不因退還而延期，其原使用期限，退還時若已逾期者，將同時失效。
• 送禮人得於禮物盒送出後，在收禮人尚未完成禮物領取前，可要求取消禮物盒，以取消本次購買的禮物商品。在禮物盒送出後至收禮人打開禮物盒之前 (最長期限為 3 天)，送禮人均可隨時透過系統操作取消禮物盒 ；若收禮人打開禮物盒後，在領取禮物或該領取期限失效前 (最長領取期限為打開禮物盒後 14 天)，送禮人僅能透過客服人工取消禮物盒。請注意，因客服的作業會有時間差距，因此透過客服人工取消禮物盒是有可能因過程中收禮人已完成禮物領取而無法取消成功的。
• 本條款中有關天數的計算，均以前提事件發生後之翌日起算第 1 日，至末日之 23:59:59 為止。
• 每份禮物限單次使用完畢，不可拆分或部分退款。

三、Fun 幣政策
• Fun 幣存續期限六個月，可依每次消費遞延，六個月內未消費將失效歸零。若消費時忘記使用 Fun 幣折扣，Fun 幣仍可保留至下次消費折抵使用。
• 若因不可抗力因素導致訂單取消，訂單所使用的 Fun 幣將會全額退還至 FunNow 帳戶。
• 訂單金額超過 TWD$100 ，可使用 Fun 幣折抵，單筆交易最高折扣為 100 Fun 幣。
• 你必須同意並遵守 Fun 幣使用規定。如本公司發現你以不當行為獲得 Fun 幣或將 Fun 幣轉賣第三者使用或有其它營利、投機取巧的行為，對於第三者商品或服務的爭議，本公司不對其負有任何形式賠償或補償的責任或義務，且本公司將擁有暫停或終止你所有服務的使用權，如果致本公司受有損害，將不排除依法追究。

如何獲取 Fun 幣
• 消費評論享 1% 無上限 Fun 幣回饋。
• 好友加入 FunNow ，輸入好友邀請碼後，受邀者首次消費，雙方都可獲得 100 Fun 幣。
